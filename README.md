# Cours de Programmation Python

Bienvenue sur le dépôt officiel du cours de programmation en Python. Ce projet a pour but de centraliser et de rendre accessible l'ensemble du matériel pédagogique destiné aux étudiants.

Ce cours est hébergé en ligne grâce à GitHub Pages pour un accès simplifié. Vous pouvez consulter les leçons directement depuis votre navigateur, sans avoir besoin de télécharger les fichiers.

## Comment Contribuer ou Signaler un Problème

Si vous êtes un étudiant et que vous constatez une erreur, une coquille ou un problème d'affichage, le moyen le plus simple de le signaler est d'ouvrir une **"Issue"** directement sur ce dépôt GitHub.

1.  Cliquez sur l'onglet "Issues" en haut de la page.
2.  Cliquez sur "New Issue".
3.  Donnez un titre clair à votre signalement et décrivez le problème en détail.

Votre participation aide à améliorer la qualité du cours pour tous !
